# ИИкона — руководство пользователя

Настройка и эксплуатация подсистем ИИконы — AI-коннектора для 1С:Предприятие 8.

Раздел для пользователей и администраторов: как настроить и запустить.

## Подсистемы

- **[Анализ ошибок](анализ-ошибок/README.md)** — мониторинг ошибок, дашборд, журнал и
  **[аудитор кода](анализ-ошибок/аудитор-кода.md)** — диагностика по стеку через LLM.
- **[Настройка моделей](настройка-моделей/README.md)** — коннектор ИИ, провайдеры,
  форматы API, авторизация, прокси, модели-эмбеддеры.
- **[RAG](rag/README.md)** — семантический поиск по базе знаний, egress-политики,
  внутренний и внешний бэкенды.
- **[MCP-сервер](mcp-сервер/README.md)** — база 1С как набор инструментов для внешних
  ИИ-агентов: настройка, авторизация, безопасность, аудит.
- **[Диаграммы](диаграммы/README.md)** — генератор диаграмм по описанию или коду,
  форматы, рендеринг через Kroki.

## Условные обозначения

Плейсхолдеры в командах — `<хост-движка>`, `<порт>`, `<путь-к-дампу>` — подставьте свои значения.
